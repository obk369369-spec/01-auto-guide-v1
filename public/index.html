자동화 안내서 Word 템플릿 다운로드

위 버튼을 눌러 잠금이 걸린 자동화안내서_v1.1.dotx 템플릿을 내려받아 고객 안내서 기본 틀로 사용합니다.

1단계. 고객 데이터 불러오기

엑셀 파일이 아직 정리가 덜 되어 있어도, 일단 파일을 선택해서
“고객 데이터 불러오기” 버튼만 눌러 두면 준비 상태를 기록해 둡니다.

2단계. 고객 데이터 분석 요약

분석 로직은 나중에 엑셀 열 구조가 확정되면 안에 넣고,
지금은 어떤 기준으로 분류·우선순위화 할지 메모용으로 사용합니다.

3단계. 안내서 초안 자동 생성

아래 버튼을 누르면 기본 텍스트 보고서를 만들고,
필요하면 워드 템플릿을 열어서 붙여 넣어 쓸 수 있습니다.

자동 생성 안내서 (복사해서 사용)

4단계. 고객 반응 · 후속조치 기록

전화 / 추가자료 메일 / 무응답 등 고객 반응을 선택하고
후속조치 내용을 메모로 남기면, 나중에 엑셀·로그로 내보내는 기반이 됩니다.
